# webtrc-voice-chat

group voice chat. based on [pion](https://github.com/pion/webrtc) webrtc go library.

# features

- mute/unmute microphone
- mute/unmute speaker
- to join a room write anything after slash e.g `/myroom` `/123` `/test` etc
